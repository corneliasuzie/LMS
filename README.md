# LMS - Prototype

Project minimal pour démonstration (frontend: HTML/CSS/JS, backend: PHP/MySQL).

Quickstart:
```bash
# Démarrer XAMPP
sudo /opt/lampp/lampp start

# Importer la base
sudo /opt/lampp/bin/mysql -u root -S /opt/lampp/var/mysql/mysql.sock < sql/migrate.sql

# Ouvrir dans le navigateur
http://127.0.0.1/lms/
```

Fichiers : voir `DOCUMENTATION_FR.md`.
